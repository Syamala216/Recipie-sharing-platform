 Recipe Sharing Platform

 About the Project
The Recipe Sharing Platform is a Django web application where users can add, view, and share recipes with images, ingredients, and descriptions.  
It helps users manage their recipes and discover new cooking ideas easily.

---

 Features
- ➕ Add recipes (title, description, ingredients, image)
- 📖 View all recipes
- 🖼 Upload recipe images
- 🔍 Search and browse recipes
- 🗂 Simple and clean UI
- ⚙ Admin panel to manage data

 How to Run

1. Clone the repository:
git clone https://github.com/Syamala216/Recipie-sharing-platform.git

2. Go inside folder:
cd Recipie-sharing-platform

3. Install Django:
pip install django

4. Run migrations:
python manage.py migrate

5. Start server:
python manage.py runserver

6. Open in browser:
http://127.0.0.1:8000/
